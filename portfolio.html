Bamazon is a CLI app designed to mimick a shopping app. The customer can choose a product and the quantity, and with the help of MySQL, the new quantity will be updated after the customer is finished shopping.

There's also an option to run the manager's side of the application. The user will be prompted with either viewing the products, viewing low inventory products, adding quantities to already existing products, or adding new products to the inventory.

Javascript, NodeJS, and MySQL were used.

Friend Findermore_vert

Link Here

close

A friend finder application based on characters from Rick and Morty. Users will enter their name and add an image to the form field before answering ten questions in order to find their closest compatible match. The user's information and answers will also be stored in JSON format in the API Friends List page.

The best match will be determined by adding the user's scores and comparing them to the default character's scores. The closest scores are matched.

HTML, CSS, Bootstrap, Javascript, NodeJS were used along with Express and Path dependencies.
