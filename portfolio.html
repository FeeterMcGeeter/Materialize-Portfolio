Bamazon CLI Appmore_vert

Link to GitHub

close

Bamazon is a CLI app designed to mimick a shopping app. The customer can choose a product and the quantity, and with the help of MySQL, the new quantity will be updated after the customer is finished shopping.

There's also an option to run the manager's side of the application. The user will be prompted with either viewing the products, viewing low inventory products, adding quantities to already existing products, or adding new products to the inventory.

Javascript, NodeJS, and MySQL were used.

Class Project 2more_vert

Link to Project

close

This project was created for my second class project. It is a social messaging site designed around socket.io. We used sockets to create a connection between the client and the server. Passport.js was used for authentification for user sign up and log in purposes. In the future, we will be adding functionality for the like button that is dynamically added with each message a user posts, along with a way to create additional chat rooms to keep load times down.

Technologies used were Javascript, Node.js, socket.io, express, and Sequelize/MySQL. Express-handlebars was used to add the HTML and Materialize for the CSS.
